# Fight-Championship

## Configuração do Ambiente

Este projeto utiliza Python com Streamlit. Para executá-lo, você precisa ter o ambiente conda `Fight_New` configurado com as seguintes dependências:
- Python 3.9+
- NumPy
- Pandas
- Streamlit

## Como Executar

1. Ative o ambiente conda:
```bash
conda activate Fight_New
```

2. Execute a aplicação:
```bash
streamlit run main.py
```

A aplicação será iniciada automaticamente no seu navegador padrão.
